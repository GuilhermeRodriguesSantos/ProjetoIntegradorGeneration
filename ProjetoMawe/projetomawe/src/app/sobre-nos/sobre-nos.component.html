Sobre nós:

Criamos conexões para demonstrar que há outras maneiras de criar e consumir, sem gerar grandes impactos a natureza.

Objetivos:

Estimular o reuso de materia prima descartada. Incentivar por materiais biodegradáveis. Equalizar a diferença de valor de produtos sustentáveis.

Nosso Ideal:

Como uma maneira de reagir ao descarte de materiais que se acumulam na atmosfera. A ideia do e-commerce, é incentivar empresas a criar soluções e o meio para apresentar aos usuários; classificamos os produtos

Linguagens de Programação/Estilização

HTML5

Usamos o HTML5 commo ferramenta, justamente com a finalidade de mostrar toda a escrita da nossa plataforma, conforme aprendizado nas aulas.

CSS3

O CSS3 é uma ferramenta de estilização, onde usamos ele em tamanho de imagem, fonte, background para deixar uma experiência melhor para o nosso usuário.

Javascript

O Javascript foi usado como uma ferramenta para colocar toda nossa parte logica da plataforma, inclusive o botão dark foi usado com ele.

Java

A linguagem de programação Java foi usada com o Eclipse IDE, a finalidade dele foi usada justamente para criar um vinculo com o Sprinboot.

Bootstrap

O Bootstrap foi usado para facilitar a programação e ajudar a deixar nossa plataforma mais bonita, usando cards, carrouseis, jumbotrons e afins.

Springboot

O Spring é uma ferramenta muito util que nos ajudou a ter um vinculo com a plataforma, Mysql(banco de dados) usando o Eclipse IDE.
